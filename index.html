WORQSHOP

... is a small development environment for the iPad. It features a fast code editor with syntax highlighting and GitHub commit support.

The editor is implemented using native iOS APIs like Core Text. It is fast, responsive, and it can edit large files, for example, a 10000-line source file. It features syntax highlighting for Python and Ruby files.

Worqshop can import projects from GitHub and commit modified files to the repositories. Use it to update your web content on GitHub Pages. Use it to edit your project code files on the iPad and set up GitHub webhooks for automatic deployment and so forth. Easy.



 

Features

Code editor

  • Written using native iOS APIs, e.g. Core Text.
  • Edit small or large files, e.g. 10000-line files.
  • Syntax highlighting support for Python, Ruby, HTML, CSS, and JavaScript files.
  • Fast and responsive.
  • Easy access to multiple files with editor tabs.
  • Autosave content during an editing session.
  • Standard editing commands such as cut, copy, and paste.
  • Type using the on-screen keyboard or an external keyboard.

GitHub integration

  • Import GitHub projects to your iPad.
  • Support for private and public repositories.
  • Commit modified files to GitHub repositories.
  • Show the status of files, whether they are modified or not.

Miscellaneous

  • The library view that shows the programming language reference.

FAQs

What are the system requirements?

Worqshop requires an iPad with iOS 5 and a GitHub account.

Who are the Worqshop users?

Worqshop is created for solo developers. At the moment, it is not suitable for developers working in a team.

Does Worqshop support other GitHub operations (e.g. merge, pull) or features (e.g. issues)?

Not yet. But they may be added in future updates.

How about themes, line number, or syntax highlighting for other files?

They are not there yet. Let us know of new features that you want. We will try to incorporate them in future updates.

Can you lower the price?

No, the price covers future updates, e.g. new features or bug fixes, for version 1 of Worqshop; and you will get the updates for free.

How do I contact you for feedback or bug reports?

Our contact details are listed on our site or you can follow us on Twitter.
